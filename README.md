# IT Support & Technical Support Engineer Portfolio

## Overview

This repository contains hands-on IT Support and Technical Support projects completed in a virtual lab environment.

The objective of these projects was to gain practical experience in system administration, troubleshooting, user management, networking, incident handling, and technical documentation.

---

## Sample Troubleshooting Workflow

1. Receive user incident.
2. Gather information and reproduce the issue.
3. Identify root cause through logs, services, or network diagnostics.
4. Apply corrective action.
5. Verify resolution.
6. Document findings and close the ticket.
